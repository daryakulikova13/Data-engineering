[Результат выполнения скрипта](screenshot.jpg)

# Описание скрипта

Скрипт `data_loader.py`:
- Загружает CSV файлы из Google Drive по ID
- Выводит информацию о данных (размер, колонки)
- Показывает первые 10 строк для проверки
